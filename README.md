# 🏥 Hospital Emergency Dashboard (Power BI Project)

## 📌 Project Overview  
This project focuses on analyzing hospital emergency room data using Power BI to gain meaningful insights into patient flow, wait times, and hospital performance. The dashboard helps in better decision-making by visualizing key metrics in an interactive way.

## 🎯 Objectives  
- Monitor patient inflow and outflow  
- Analyze waiting time trends  
- Track admission vs non-admission cases  
- Identify peak hours and busiest days  
- Improve hospital efficiency using data-driven insights  

## 📊 Dashboard Features  
- 📈 Total Patients Overview  
- ⏱️ Average Waiting Time Analysis  
- 🚑 Admission vs Non-Admission Breakdown  
- 📅 Daily & Monthly Patient Trends  
- 🕒 Peak Hour Analysis  
- 👨‍⚕️ Department-wise Patient Distribution  

## 🛠️ Tools & Technologies Used  
- Power BI – Data Visualization  
- Excel / CSV – Data Source  
- Power Query – Data Cleaning & Transformation  
- DAX – Calculations & Measures  

## 📂 Dataset Description  
The dataset contains hospital emergency room data including:  
- Patient ID  
- Admission Date & Time  
- Waiting Time  
- Department  
- Admission Status  
- Gender & Age  

## 🚀 Key Insights  
- Identified peak hours with maximum patient load  
- Found patterns in waiting time delays  
- Analyzed department efficiency  
- Understood patient admission trends
